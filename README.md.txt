# Descomplicando a Inteligência Artificial na Prática - Landing Page

Esta é a landing page oficial do e-book **"Descomplicando a Inteligência Artificial na Prática"**, criado por Eduardo Serafim. Aqui você encontrará uma apresentação clara, persuasiva e responsiva do conteúdo, ideal para divulgar o material e aumentar as conversões de vendas.

## 🚀 Sobre o E-book
Um guia prático feito para iniciantes que querem entender e aplicar a inteligência artificial no dia a dia, no trabalho, nos estudos ou nos negócios, sem complicações técnicas.

## 🧱 Estrutura da Página
- **HTML5 e CSS3 responsivo**
- Seções otimizadas para conversão:
  - Cabeçalho com promessa
  - Benefícios claros
  - Público-alvo
  - Sobre o autor
  - Bônus
  - Chamada para ação (CTA)
- Totalmente editável e personalizável

## 📸 Imagem de Capa
Inclua a imagem `capa.jpg` na mesma pasta que o HTML para que ela seja exibida corretamente na seção de destaque.

## 🧾 Como Visualizar Localmente
1. Faça o download ou clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/ebook-inteligencia-artificial.git
   ```
2. Abra o arquivo `index.html` em seu navegador.

## 🌐 Publicar no GitHub Pages
1. Suba os arquivos para seu repositório no GitHub.
2. Vá em **Settings > Pages**.
3. Em "Source", selecione a branch `main` e o diretório `/root`.
4. Clique em "Save".
5. Seu site será publicado em: `https://seu-usuario.github.io/nome-do-repositorio`

## 📬 Link de Compra
No botão da landing, substitua o trecho abaixo pelo seu link de pagamento real (ex: Mercado Pago, Hotmart, etc):

